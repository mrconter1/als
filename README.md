# ALS - Accessible Keyboard System

A simple on-screen keyboard designed to help people with disabilities type efficiently using binary search navigation.

## How It Works

**main.py** - Interactive keyboard interface where you navigate keys using spacebar binary search or click directly. Includes text-to-speech output (SAY button) and basic editing controls.

## Features

- Binary search navigation through keys
- Direct mouse click selection
- Speak text aloud
- Auto-capitalization after punctuation
- Basic text editing (backspace, clear, cursor movement)
